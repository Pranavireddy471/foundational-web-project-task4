# Responsive Portfolio Web Application

## 📌 Project Overview

This project is a **responsive web application** developed using **HTML, CSS, and JavaScript**.
It combines multiple front-end development concepts into a single application, including a **personal portfolio website, a to-do list with local storage, and a product listing page with filtering and sorting features**.

The application demonstrates how to create interactive and responsive user interfaces using modern web technologies.

---

## 🎯 Objectives

* Build a **professional personal portfolio website**.
* Implement a **To-Do List application with Local Storage** for persistent data.
* Create a **Product Listing Page with filtering and sorting options**.
* Apply **responsive web design using CSS media queries**.

---

## 🛠 Technologies Used

* **HTML5** – Structure of the web pages
* **CSS3** – Styling and responsive design
* **JavaScript (ES6)** – Interactivity and functionality
* **Local Storage API** – Data persistence for To-Do tasks

---

## ✨ Features

### 1. Personal Portfolio Website

The portfolio section includes:

* **Home Section** – Introduction to the portfolio
* **About Section** – Information about skills and education
* **Projects Section** – List of sample projects
* **Contact Section** – Contact form for communication

The website includes a **navigation menu** that allows users to scroll smoothly between sections.

---

### 2. To-Do List Application

The To-Do List allows users to manage tasks efficiently.

Features include:

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Tasks are saved using **Local Storage**
* Tasks remain available even after refreshing the page

---

### 3. Product Listing Page

This section displays products dynamically.

Each product contains:

* Product name
* Category
* Price
* Rating
* Image

Users can interact with the product list by:

**Filtering**

* Filter by category (Electronics / Clothing)

**Sorting**

* Price (Low → High)
* Price (High → Low)
* Rating

The products update dynamically using **JavaScript DOM manipulation**.

---

## 📱 Responsive Design

The website is fully responsive and works across different devices.

CSS **Media Queries** are used to adjust:

* Layout structure
* Navigation menu
* Product grid layout

### Breakpoints Used

* **Desktop:** Above 900px
* **Tablet:** 600px – 900px
* **Mobile:** Below 600px
